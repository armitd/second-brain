# Role Pack: Enterprise Architect

## Recommended Skills

| Skill | Why it matters for you |
|---|---|
| knowledge-consolidation | Turn scattered architecture decisions, pattern notes, and tech evaluations into living reference frameworks and decision records |
| braindump | Capture design rationale, tradeoff analysis, and system observations on the fly with automatic classification |
| daily-brief | Stay current on cloud platforms, security advisories, industry standards, and technology shifts across your enterprise stack |
| comprehensive-analysis | Deep analysis for architecture reviews, technology radar updates, and strategic planning sessions |
| meeting-transcript | Capture architecture review board decisions, design workshops, and stakeholder alignment outcomes with structured tracking |
| weekly-checkin | Spot cross-domain patterns — recurring integration issues, platform adoption trends, technical debt accumulation |
| auto-research | Multi-agent parallel investigation for technology evaluations, vendor comparisons, and standards research |
| url-dump | Save architecture references, RFCs, cloud provider updates, and industry whitepapers with auto-extracted insights |
| team-brief | Cross-reference GitHub, Linear, and Slack to understand what teams are building and where architecture guidance is needed |
| generate-prd | Draft architecture decision records (ADRs) and technical specifications with approval gates |
| onboarding | Initial setup — run this first |
| update-cog | Keep your COG framework current |

## Recommended Integrations

| Integration | Why it matters for you |
|---|---|
| github | Architecture-significant PRs, infrastructure changes, and cross-repo dependencies |
| slack | Architecture channel discussions, incident threads, and cross-team coordination signals |
| linear | Track architecture-related work items and ensure design decisions flow into implementation |
| confluence | Publish and maintain architecture documentation, ADRs, and reference architectures |

## Suggested Agent Mode

`team` — Enterprise architects work across enough domains and data streams that delegated analysis produces significantly better results, especially for comprehensive-analysis, auto-research, and knowledge-consolidation.

## Notes

- **Knowledge consolidation is your superpower** — scattered architecture notes across meetings, braindumps, and research naturally evolve into reusable patterns, reference architectures, and decision frameworks in `05-knowledge/`.
- **Use projects (`04-projects/`) for active architecture initiatives** — migration programs, platform modernization, new capability rollouts each get their own tracking space.
- **The daily brief surfaces signals across your entire technology landscape** — configure your interests around cloud providers, security, standards bodies, and key vendors in your enterprise stack.
