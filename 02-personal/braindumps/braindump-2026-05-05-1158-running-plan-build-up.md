## Strategic Intelligence

### Key Insights
1. **The treadmill removes the biggest beginner barrier** — weather, safety, and convenience excuses are gone. Use this window while the novelty is high to build the habit before motivation normalises.
2. **The enemy of building up is going too fast too soon** — the most common reason beginners quit or get injured is running at a pace that feels "like running" when the aerobic system isn't ready. On a treadmill, you control pace exactly — use this.
3. **Build time/minutes first, distance second** — distance is a vanity metric at the start. What you're building is aerobic capacity and connective tissue tolerance. Time on feet is the right unit.

### Suggested Build-Up Framework

A proven progressive approach for treadmill from zero:

**Phase 1 — Foundation (Weeks 1–3): Walk/run intervals**
- 3× per week, 20–25 minutes per session
- Week 1: 1 min run / 2 min walk, repeat 6–7×
- Week 2: 90 sec run / 90 sec walk, repeat 7–8×
- Week 3: 2 min run / 1 min walk, repeat 6–7×
- Target pace: conversational — you should be able to talk in full sentences while running

**Phase 2 — Extension (Weeks 4–6): Longer run blocks**
- 3× per week, 25–30 minutes
- Week 4: 5 min run / 2 min walk, repeat 3–4×
- Week 5: 8 min run / 2 min walk, repeat 3×
- Week 6: 12 min run / 2 min walk, repeat 2×

**Phase 3 — Continuous (Weeks 7–10): Building to sustained running**
- 3× per week
- Week 7: 20 min continuous easy run
- Week 8: 25 min continuous
- Week 9: 28 min continuous
- Week 10: 30+ min continuous — by this point you're at or near 5K depending on pace

**Target pace guidance for treadmill:**
- Easy/conversational: 7.0–8.5 km/h (depending on fitness)
- Not: the pace that makes you feel like you're working hard — that comes later
- A useful check: if you can't say a full sentence out loud, slow down

### Pattern Recognition
- This is a classic high-motivation + low-baseline start — the risk window is weeks 2–4 when initial enthusiasm meets discomfort before fitness arrives. Build the routine before the novelty fades.
- Treadmill has a unique advantage: incline. Even a 1% incline better replicates outdoor running biomechanics and is worth setting as your default.

### Strategic Implications
- If there's an event or target date in mind, work backwards from it — this framework gets most people to 5K in 8–10 weeks from zero
- Consistency (3× per week) beats intensity at this stage — one longer run per week is less effective than three shorter ones
