## Strategic Intelligence

### Key Insights
1. **The treadmill removes the biggest beginner barrier** — weather, safety, and convenience excuses are gone. Use this window while the novelty is high to build the habit before motivation normalises.
2. **The enemy of building up is going too fast too soon** — the most common reason beginners quit or get injured is running at a pace that feels "like running" when the aerobic system isn't ready. On a treadmill, you control pace exactly — use this.
3. **Build time/minutes first, distance second** — distance is a vanity metric at the start. What you're building is aerobic capacity and connective tissue tolerance. Time on feet is the right unit.

### Suggested Build-Up Framework — Zero to 10K (16–18 weeks)

Target: run 10K continuously. Three sessions per week throughout.

**Phase 1 — Foundation (Weeks 1–3): Walk/run intervals**
- 3× per week, 20–25 minutes per session
- Week 1: 1 min run / 2 min walk, repeat 6–7×
- Week 2: 90 sec run / 90 sec walk, repeat 7–8×
- Week 3: 2 min run / 1 min walk, repeat 6–7×
- Target pace: conversational — you should be able to talk in full sentences while running

**Phase 2 — Extension (Weeks 4–6): Longer run blocks**
- 3× per week, 25–30 minutes
- Week 4: 5 min run / 2 min walk, repeat 3–4×
- Week 5: 8 min run / 2 min walk, repeat 3×
- Week 6: 12 min run / 2 min walk, repeat 2×

**Phase 3 — Continuous base (Weeks 7–10): Building to 30 min non-stop**
- 3× per week
- Week 7: 20 min continuous easy run
- Week 8: 25 min continuous
- Week 9: 28 min continuous
- Week 10: 30 min continuous — approx 4–4.5K at easy pace, solidly aerobic

**Phase 4 — Distance extension (Weeks 11–15): Long run strategy**
- Keep 2 sessions per week at 30 min easy
- Add 1 long run per week, building as follows:
  - Week 11: 35 min long run
  - Week 12: 40 min long run
  - Week 13: 45 min long run
  - Week 14: 50 min long run
  - Week 15: 55–60 min long run — at easy pace (7.0–8.0 km/h) this is ~7–8K

**Phase 5 — 10K reach (Weeks 16–18):**
- Week 16: 65 min long run — first time near 10K distance
- Week 17: consolidate — repeat 60 min feeling controlled
- Week 18: 10K attempt — set treadmill to 10.0km and run it

**Target pace guidance for treadmill:**
- Easy/conversational: 7.0–8.5 km/h (depending on fitness)
- Long run pace: the low end of your easy range — you should feel like you could run for a very long time
- A useful check: if you can't say a full sentence out loud, slow down
- Set treadmill incline to 1% as your default — better replicates outdoor running biomechanics

### Pattern Recognition
- This is a classic high-motivation + low-baseline start — the risk window is weeks 2–4 when initial enthusiasm meets discomfort before fitness arrives. Build the routine before the novelty fades.
- The bigger risk for 10K is weeks 11–15 — long run sessions start to feel like a real time commitment (45–60 min). Schedule them on a day with flexibility.
- Connective tissue (tendons, ligaments) adapts slower than aerobic fitness — this is why the plan doesn't rush. Injury at week 10 from going too hard is the most common reason people never reach 10K.

### Strategic Implications
- From today (5 May 2026), 18 weeks puts the 10K attempt at ~10 September 2026
- Consistency (3× per week) beats intensity — never skip the easy sessions to save time for one hard one
- If there's a 10K event to target, that's a powerful motivator — parkrun 10K events or a local race in September would align well
