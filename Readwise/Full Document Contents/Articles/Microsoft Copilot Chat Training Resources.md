I have had a few customers ask for some Copilot Chat Only training this week. While there is a lot out there, here are a few resources I have been sharing.

I would love to know some of your favorite learning modules, tips, tricks, to continue to add to mine!!??

Explore Microsoft 365 Copilot Chat (Microsoft Learn)  

Covers:  

What Copilot Chat is  

How to access it  

How grounding works  

How to write better prompts  

How to work with files, agents, and notebooks

Link: [https://lnkd.in/g5FEqrTW](https://lnkd.in/g5FEqrTW?trk=public_post-text) [[learn.microsoft.com](http://learn.microsoft.com?trk=public_post-text)]

Copilot Chat – Prompting & Productivity  

Write effective prompts to achieve optimal results  

Core guidance on prompt structure (context, goal, format), common mistakes, and using the Prompt Gallery.
