# Thread: 7 Claude Prompts for Parallel Work at Scale

## Summary
A prompt library for using Claude as a parallel work system — seven structured prompts for spawning agent teams, auditing codebases, maintaining context memory across sessions, adaptive problem solving, mistake spotting, long-term project tracking, and cross-repo integration. Low-effort but useful as a reference prompt template library.

## Key Points
- **Parallel Work Manager:** Spawn agent teams per task, coordinate, merge results
- **Multi-File Auditor:** Upload project folder, get prioritised action plan covering tech debt, security, performance
- **Context Memory Vault:** Structured context block at session start — tell Claude everything it needs to retain, confirm understanding
- **Adaptive Problem Navigator:** Ask Claude to assess reasoning depth needed before answering, not after
- **Mistake Spotter:** Explicitly ask Claude to challenge your assumptions and find logical errors mid-conversation
- **Long-Term Project Assistant:** Use context compaction explicitly for month-long projects

## Why It Matters
The Mistake Spotter and Adaptive Problem Navigator prompts are immediately useful for professional braindump sessions and architecture review work — prompts that force Claude to challenge rather than confirm are higher-value than clarification prompts.
