# Boost Annotation Speed 5x With Model Accelerated Annotation

## Summary
Covers model-accelerated annotation (semi-supervised learning) as an alternative to manual image labelling. Rather than annotating every image manually, the model generates suggestions and human annotators validate or correct them — 5x speed improvement. Directly relevant to building training datasets for computer vision applications like vehicle damage classification.

## Key Highlights
- "When building computer vision applications, the quality of your data is everything"
- Traditional annotation is time-consuming and resource-intensive at scale
- Model-accelerated annotation = human validates machine suggestions rather than labelling from scratch
- Semi-supervised learning enables faster, more efficient dataset scaling

## Why It Matters
If the damage assessment PoC moves toward a custom model (rather than pure foundation model API calls), benchmark test set annotation will be a bottleneck. This technique is directly applicable to building the training/evaluation dataset for vehicle glass damage types. Worth sharing with whoever leads the technical PoC work.
