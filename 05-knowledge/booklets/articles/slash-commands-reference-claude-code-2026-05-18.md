# Slash Commands Reference — Claude Code

## Summary
Official Claude Code reference guide for slash commands — the keyboard shortcuts and directives that control planning, reviewing, automating, and managing coding tasks. Covers cost-related warnings and best-use patterns for commands including `/goal`, `/loop`, and `/autofix-pr`. The key insight: combining `/goal` + `/loop` lets Claude work autonomously toward a clear, measurable target.

## Key Highlights
- `/goal` + `/loop` combination enables autonomous task completion — Claude iterates until the goal is met
- `/loop` without an interval lets the model self-pace its work
- `/autofix-pr` automates PR review and fix cycles in CI workflows
- Cost warnings built in — some commands trigger expensive multi-step operations
- Planning and review commands separate the "think" phase from the "act" phase

## Why It Matters
Directly relevant to how COG's own scheduled agents and remote tasks work — understanding slash commands is the foundation for building more sophisticated automated workflows. Also relevant to the MCP Governance project when thinking about how agentic workflows are structured and controlled.
